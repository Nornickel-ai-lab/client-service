# client-service

Vue чат на shadcn-vue + Tailwind v4.

## Разворот

```bash
cd ../server-service
task up
```

http://localhost:8080

## Локально

```bash
npm install
npm run dev
```

## Компоненты

shadcn-vue в `src/shared/ui/`  
строки UI в `src/shared/config/ui.ts`

## Сборка

```bash
npm run build
```
